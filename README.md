## 🚀 Running Frontend
To run locally, run the following commands to run node modules and serve the website locally.
```bash
cd ./client
```

```bash
yarn install
```

```bash
yarn run dev
```
## 🚀 Running Backend
To run locally, run the following commands to run node modules and serve the website locally.
```bash
cd ./server
```

```bash
npm i
```

```bash
node index.js
```
